# BASE DEFENSE

A fast-paced, neon-themed tower defense game built for the Farcade platform.  
Optimized for mobile play and fully integrated with the Farcade Game SDK.

## Features
- Auto-aim tower shooting system  
- Enemy waves scaling dynamically  
- Upgrades between waves  
- Hologram FX, neon theme  
- Integrated Farcade ready/gameOver/mute/playAgain
- 60 FPS canvas rendering

## How to Run
Just open `index.html` in any browser  
or upload the project to Remix / Farcade / GitHub Pages.
